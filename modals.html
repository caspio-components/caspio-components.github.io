Note to all Developers
  • Add the following links to your page:
    <link rel='dns-prefetch' href='//fast.wistia.net' />
    <script type='text/javascript' src='https://fast.wistia.net/assets/external/E-v1.js?ver=1'></script>
  • Please make sure that you link the Bootstrap 3.3.7 js file at the bottom of the <body> tag.
  • Check if the Social media link is correct.
  • If there are multiple Wistia video popup on a single page, please make sure to update the ID and VIDEO ID:

    <div class="wistia_embed wistia_async_lqdusrvh9q videoFoam=true" style="width:100%;" id="my_video"> </div>

Wistia Options name Values Descriptions
autoPlay true The video will play as soon as it’s ready.
false This is Default
controlsVisibleOnLoad true Controls like the big play button, playbar, volume, etc. will be visible upon page load
false This is Default
copyLinkAndThumbnailEnabled true This is Default
false Once your video is embedded on a webpage, the option to “Copy Link and Thumbnail” when you right click on your video will be removed.
You will not be able to create a thumbnail that links to the page where the video is embedded.
doNotTrack true Data for each viewing session is tracked and reported back to the Wistia servers for display in heatmaps and aggregation graphs.
false This is Default.
email <email> Associate an email with this video’s viewing session. This is equivalent to running video.email(email) immediately after initialization.
endVideoBehavior default The video stays on the last frame.
reset The video shows the thumbnail and controls if they are set to be visible on load.
loop The video plays again from the beginning.
fullscreenButton true The fullscreen button will be available as a video control.
false This is the Default
muted true The video will start in a muted state.
false This is the Default
playbackRateControl true This is the Default
false The playback speed controls in the settings menu will be hidden.
playbar true The playbar — which includes the playhead, current time, and scrubbing functionality — will be available
false It is hidden
playButton true The big play button control will appear in the center of the video before play.
false This is the Default
playerColor #ffffff Sets the player's theme color.
playlistLinks <URL> This option lets you associate specially crafted links on your page with a video, turning them into a playlist.
playlistLoop true It will embed the first video once the last video has finished.
false This is Default
qualityMax 224, 360, 540, 720, 1080, 3840 This allows you to specify the maximum quality the video will play at. Wistia will still run bandwidth checks to test for speed, and play the highest quality version at or below the set maximum.
settingsControl true The settings control will be available. This will allow viewers to control the quality and playback speed of the video.
false This is Default
smallPlayButton true The small play button control will be available.
false This is the Default
volume 0 Mutes the video on load
1 Set the volume of the video
volumeControl true A volume control is available over the video.
false This is the Default
buttonBackground #ffffff The hex color of the button. Defaults to the player color.
autoSize true sets a font-size and line-height for the call to action based on the height of the video.
false This is the Default
